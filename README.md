# UK Inflation Dynamics: Econometric Analysis

Independent econometrics research project analysing UK inflation dynamics between 2015–2025 using R and time-series modelling techniques.

## Objectives
- Analyse key drivers of UK inflation
- Evaluate post-COVID inflation persistence
- Assess labour market and energy price effects
- Forecast inflation behaviour using econometric models

## Methods
- Multiple Regression Analysis
- ARIMA Forecasting
- Stationarity Testing
- Time-Series Econometrics
- Autocorrelation Diagnostics

## Key Findings
- Wage growth became significantly more inflationary during the post-COVID period
- Energy prices contributed substantially to inflation volatility
- Inflation persistence strengthened during periods of macroeconomic instability

## Tools Used
- R
- ggplot2
- forecast package
- ONS, OECD, and FRED macroeconomic data
